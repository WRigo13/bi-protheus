FAQ

Perguntas frequentes.

Preciso ter o Power BI Pro para usar os dashboards? +
Para visualizar os dashboards você precisa de uma conta Power BI (gratuita ou Pro). Para publicação com atualização automática via DirectQuery, o Gateway requer acesso ao servidor — o que geralmente já existe na infraestrutura da empresa.
O projeto funciona com qualquer versão do Protheus? +
Sim. Trabalhamos diretamente com o banco de dados (SQL Server ou Oracle), independente da versão do Protheus. O que importa é o acesso ao banco — não à versão do ERP.
Quanto tempo leva para entregar um dashboard? +
Dashboards simples (1-2 visões, dados já mapeados): 1 a 2 semanas. Projetos completos como os do portfólio (5 painéis integrados): 4 a 6 semanas, incluindo diagnóstico, modelagem e treinamento.
Qual é o investimento? +
Depende do escopo. Projetos simples partem de R$ 2.500. Implantações completas com múltiplos painéis e DirectQuery ficam entre R$ 8.000 e R$ 18.000. O diagnóstico inicial é gratuito e sem compromisso — e já ajuda a dimensionar o projeto com precisão.
Vocês dão suporte após a entrega? +
Sim. Oferecemos suporte pós-entrega e contratos de manutenção mensal para ajustes, novos campos e evolução dos painéis conforme a operação muda.
